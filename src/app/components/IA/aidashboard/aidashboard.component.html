❓ Comprendre l'IA - Guide simple

🤖

Qu'est-ce qu'un modèle IA ?

C'est un programme qui apprend à partir de vos données pour faire des prédictions. Comme un expert qui devient meilleur avec l'expérience.

📊

Précision des prédictions

Un score de 85% signifie que l'IA a raison 85 fois sur 100. C'est excellent pour la gestion de stock !

⚠️

Types d'anomalies

  • Stock critique : Il ne reste presque plus rien
  • Stock dormant : Beaucoup de stock qui ne bouge pas
  • Rotation élevée : Le stock s'écoule très vite
💡

Comment utiliser l'IA ?

  1. Cliquez sur "Entraîner l'IA" pour que le modèle apprenne de vos données
  2. Consultez les recommandations pour voir les actions prioritaires
  3. Utilisez le test de prédiction pour simuler des scénarios
  4. Scannez régulièrement les anomalies pour détecter les problèmes
